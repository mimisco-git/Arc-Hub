# ArcLume

ArcLume is a premium Arc Testnet wallet intelligence dashboard for live balance checks, wallet activity visibility, Arcscan linking, and cleaner Arc-first onchain insight.

## Features
- Live Arc Testnet wallet lookup
- Real balance and nonce loading
- Arcscan wallet and transaction linking
- Network status panel
- GitHub Pages friendly deployment
